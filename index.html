2) Explain About Multi-channel Convolutional Operation in Neural Networks

  • Multi-channel convolution is an extension of single-channel convolution.
  • It is used in CNN to process the input with multiple channels.
  • Example:
    • RGB images with 3 channels:
      • Red
      • Green
      • Blue
  • In Single Channel Convolution, it performs a dot product to generate the output.
  • In Multiple Channel Convolution, each channel has its own features.
  • Input: N channels
  • Filter Size: K x K x N
  • The convolution sums all channels and produces a single output channel.

3) Discuss about working principle and applications of GAN (Generative Adversarial Networks)

  • GAN:
    • GAN stands for Generative Adversarial Networks
  • Generator:
    • A Generator creates fake data from random inputs
    • The Goal of the Generator is to produce fake data that looks like realistic one
  • Real Data and Fake Data:
    • Real Data: Real sample from the dataset
    • Fake Data: Fake sample generated by Generator
  • Discriminator:
    • Discriminator is the binary classifier that classifies whether the data is real or fake data
  • Applications of GAN:
    • Image Generation
    • Face Generation
    • Image to Image translation
    • Enhancing Image Quality

4) Illustrate How to Use Restricted Boltzmann Machines in Deep Learning

  • RBM stands for Restricted Boltzmann Machines
  • RBM mainly consist of two units:
    • visible units
    • hidden units
  • RBM trained based on the CD Algorithm
  • There are mainly two phases in RBM:
    • a) Positive Phase
      • Updates weight based on visible units
    • b) Negative Phase
      • Updates weight based on hidden units
  • Types of RBM:
    • There are mainly two types
      • a) Binary RBM
      • b) Gaussian RBM
    • a) Binary RBM:
      • Input and Hidden Units are binary variables
    • b) Gaussian RBM:
      • Input and Hidden Units follow a Gaussian Distribution

6) CNN vs RNN

CNN RNN
CNN stands for Convolutional Neural Network RNN stands for Recurrent Neural Network
It uses a convolutional layer to extract features from the data It uses a recurrent layer to process the sequential data
It processes grid-like data It processes Sequential data
Example: images, videos, etc... Example: time-series, text, audio, etc...
Input data is independent Input data is dependent
It handles fixed-length images It handles variable-length images
CNN is less complex when compared to RNN RNN is highly complex
Applications:
  • Image Recognition
  • Object Detection
 Applications:
  • time-series prediction
  • text-to-speech
